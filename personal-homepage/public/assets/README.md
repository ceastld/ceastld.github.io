# 图片资源目录

这个目录用于存放学术成果的图片资源。

## 目录结构

```
assets/
├── papers/           # 论文相关图片
│   ├── learn2control.gif    # Learn2Control 论文的 GIF 动图
│   └── ...                  # 其他论文图片
├── projects/         # 项目相关图片
└── profile/         # 个人头像等
```

## 使用说明

1. 将论文的 GIF 动图放在 `papers/` 目录下
2. 图片文件名建议使用论文的英文名或项目名
3. 支持的图片格式：GIF, PNG, JPG, WebP
4. 建议图片尺寸：240px 宽度，高度自适应
